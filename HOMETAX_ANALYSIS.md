## 🔧 Playwright 크롤러 수정 사항

### 수정 전략:
1. **홈택스 메인 페이지 접속**
   - URL: `https://hometax.go.kr`
   - 페이지 로드 대기

2. **우측 상단 메뉴 버튼 클릭**
   - 선택자: 햄버거 메뉴 (☰) 아이콘
   - 클래스명 또는 aria-label 기반 선택

3. **좌측 메뉴에서 "기타" 클릭**
   - 텍스트: "기타"
   - 또는 클래스명 기반 선택

4. **우측 영역에서 "자료실" 클릭**
   - 텍스트: "자료실"
   - 또는 링크 기반 선택

5. **자료실 페이지에서 테이블 데이터 추출**
   - 테이블 선택자: `table tbody tr`
   - 각 행에서:
     - 제목: `td:nth-child(2) a` 또는 링크 텍스트
     - 등록일: `td:nth-child(4)` 또는 마지막에서 2번째 셀
     - 첨부파일: `td:nth-child(5)` 아이콘 확인
     - 클릭 가능한 링크 추출
